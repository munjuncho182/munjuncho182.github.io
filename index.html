ANIMAL PLUS

For the support of your lovely animals

We love our friends,Animals!

Animal + (A+) is an animal monitoring device that tracks an animal’s heartbeat, movement, body temperature, environment temperature, and location. It then delivers this data to an easy to use interface wirelessly. The interface is a web page that displays graphs and alerts about the parameters mentioned above so it is easily readable by a human.
